**Step 2:** Rasterize PNG intermediates from the SVG:

```bash
mkdir -p /tmp/rewired-favicon

rsvg-convert -w 180 -h 180 static/favicon.svg -o static/apple-touch-icon.png
```

**Step 3:** Convert the 32×32 PNG to ICO using sips:

```bash
sips -s format microsoft_icon /tmp/rewired-favicon/favicon-32.png --out static/favicon.ico
```

Note: `sips` supports `microsoft_icon` format output on macOS (verified via `sips -h formats`). If the sips ICO output fails for any reason, fallback: use the 32×32 PNG directly at `static/favicon.ico` — most modern browsers accept a PNG served as `.ico`. Record the fallback in SUMMARY.md if used.
